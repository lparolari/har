\begin{abstract}
Using smartphones sensor for predicting the current activity performed by a user could be a very useful feature for most of the mobile apps out there. However, when dealing with cheap sensor and real use case scenarios, this task can easily become very challenging. A key question in particular is how to mitigate HAR impairments due to these new problem settings. In this paper, we study state-of-the-art models behaviors, and proposed solutions to mitigate problems as Orientation Independent Transform and augmenting a CNN with automatic extracted autoencoder features. Our model outperform previous state-of-the-art model tested on Heterogeneity Dataset, and demonstrate good performances on a new real use scenario collected dataset. \\
\end{abstract}
